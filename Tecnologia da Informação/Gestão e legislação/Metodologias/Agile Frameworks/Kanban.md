# Kanban

## Principles

- Real-time communication of capacity
- Transparency of work

## Kanban board

- Visual representation of work items

### Objective

- Track the progress of work

### Card/item status

- To Do
- In Progress
- Code Review
- Done

## History

- 1940-50, Japan, Toyota Production System

## Benefits

- Planning flexibility
- Shortened time cycles
- Fewer bottlenecks
- Continuous delivery
